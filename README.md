cara menjalankan program : 
1. Download repo ini
2. Buka 'index.html'

## Features
Fitur yang terdapat dalam program ini 
* Pembuatan model:
  * Garis
  * Persegi (Segiempat sama sisi)
  * Persegi panjang (tidak harus sama sisi)
  * Poligon
* Mengubah Warna 
* Melakukan edit pada gambar
* Menggeser titik kontrol
* Upload gambar 
* Save gambar
